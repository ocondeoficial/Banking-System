Sobre o Projeto

O Sistema Bancário LGV foi desenvolvido para fornecer uma solução eficiente e segura para a gestão de contas bancárias, transações financeiras e consultas de extratos. Com uma interface amigável e segura, o sistema foi projetado para facilitar a vida dos usuários e proporcionar uma experiência sem complicações ao gerenciar suas finanças pessoais e profissionais.

Com funcionalidades como criação de contas, transferências, consulta de saldo e extratos detalhados, o LGV se destaca pela facilidade de uso e confiabilidade. O sistema adota altos padrões de segurança para garantir a proteção das informações sensíveis dos usuários, utilizando criptografia avançada para todas as transações e dados registrados.

Funcionalidades do Sistema


Criação de Contas

O sistema permite aos usuários criar novas contas bancárias com um processo simples e rápido.

Transferências Bancárias

Facilita a realização de transferências entre contas bancárias de forma rápida e segura.

Consulta de Extratos

Exibe extratos bancários detalhados com todas as transações realizadas, oferecendo visibilidade total sobre as finanças.
